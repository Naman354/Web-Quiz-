# Web Quiz Project

A simple browser-based quiz application built with HTML, CSS, and JavaScript.

## Features

- Displays one question at a time
- Highlights correct and incorrect answers
- Tracks the user's score
- Lets the user restart the quiz after finishing
- Clean and responsive card-style layout

## Project Structure

- `index.html` - Main quiz page
- `Quiz.html` - Duplicate quiz page included in the project
- `script.js` - Quiz logic and question data
- `style.css` - Styling for the app

## How to Run

1. Download or clone this project.
2. Open `index.html` in your web browser.

No installation or dependencies are required.

## How It Works

- Questions are stored in the `questions` array inside `script.js`
- Each question includes four answer choices
- When a user selects an answer, the app shows whether it is correct
- After all questions are answered, the final score is displayed

## Customization

To add or edit quiz questions:

1. Open `script.js`
2. Update the `questions` array
3. Save the file and refresh the browser

You can also update colors, spacing, and button styles in `style.css`.

## Technologies Used

- HTML5
- CSS3
- JavaScript
